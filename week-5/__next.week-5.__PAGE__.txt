### About Me üíÅ

**Goal:**  Practice **variables**, **input**, and **f-strings**.

**Requirements**
- Get input for **3 variables**:

  - one for an **age**
  - one for a **favorite color**

  - the name
  - the favorite color
  - the age **next year**

name = input("Enter your name: ")

## Challenge 4 

### Random Choice üé≤

**Goal:**  
Practice **lists**, **input**, **conditionals**, and **random.randint()**.

**Requirements**
- Create a list of **3 or more items** (for example, snacks, colors, or animals)  
- Ask the user to pick a **number** from `1` to the length of your list  
- Use `random.randint()` to generate a random number between `0` and the last index of the list  
- Print **both the user's choice** and **the randomly chosen item** from the list  
- Use **if / elif / else** to handle invalid input

**Example output**
```txt
Your list: ["apple", "banana", "cherry"]
Enter a number between 1 and 3: 2
You chose: banana
The computer randomly picked: cherry
```

<Tabs items={['Hint 1', 'Hint 2']}>

<Tab>
Remember that **list indexes start at 0**, so you may need to subtract 1 from the user's number to get the correct item.
</Tab>

<Tab>
Use `random.randint(0, len(list)-1)` to pick a random index.  
Check if the user's input is valid with `if / elif / else`.
</Tab>

<Tab>
```python copy filename="random_choice.py"
import random

# Create a list of items
items = ["apple", "banana", "cherry"]

# Ask the user for a choice
user_input = int(input(f"Enter a number between 1 and {len(items)}: "))

# Check if input is valid
if 1 <= user_input <= len(items):
    print(f"You chose: {items[user_input - 1]}")
else:
    print("Invalid choice!")

# Randomly select an item
random_index = random.randint(0, len(items) - 1)
print(f"The computer randomly picked: {items[random_index]}")
```

## Challenge 5

### Raffle Game üéüÔ∏è

**Goal:**  Practice **variables, input, f-strings, lists, random numbers, conditionals, and functions**.

**Requirements**
- Create an **empty list** called `participants`  
- Write a **function** called `enter_raffle()` that:
  - Asks the user for their **name**  
  - Adds the name to the `participants` list  
- Write another **function** called `draw_winner()` that:
  - Picks a **random winner** from the `participants` list using `random.randint()`  
  - Prints a message announcing the winner using an **f-string**  
- Call `enter_raffle()` **three times** to simulate three participants  
- Call `draw_winner()` **once** at the end to announce the winner  

**Example output**
```txt
Enter your name: Alice
Enter your name: Bob
Enter your name: Charlie

The winner is: Bob! üéâ
```

<Tabs items={['Hint 1', 'Hint 2']}>

<Tab>
Use a **list** to store the names. Each time someone enters, append their name to the list.
</Tab>

<Tab>
Use `random.randint(0, len(participants)-1)` to select a random index for the winner.
</Tab>

<Tab>
```python copy filename="raffle_game.py"
import random

# List to store participants
participants = []

# Function to enter the raffle
def enter_raffle():
    name = input("Enter your name: ")
    participants.append(name)

# Function to draw the winner
def draw_winner():
    winner_index = random.randint(0, len(participants) - 1)
    winner = participants[winner_index]
    print(f"\nThe winner is: {winner}! üéâ")

# Enter three participants
enter_raffle()
enter_raffle()
enter_raffle()

# Draw the winner
draw_winner()
```

## Challenge 6

### Number Checker üî¢

**Goal:**  Practice **functions**, **input**, **lists**, **conditionals**, and **f-strings**.

<Callout type="important">
The **modulo operator `%`** tells you the **remainder** after dividing one number by another.  

- **Example 1:** `7 % 2`  
  - 7 divided by 2 equals 3 with a remainder of **1**  
  - So `7 % 2` is **1**

- **Example 2:** `8 % 2`  
  - 8 divided by 2 equals 4 with no remainder  
  - So `8 % 2` is **0**

**Tip:** The left side is the number you want to divide, the right side is the number you divide by.  
We use this to check if a number is **even** (`num % 2 == 0`) or **odd** (`num % 2 != 0`).
</Callout>

**Requirements**
- Write a **function** called `check_number()` that:
  - Asks the user to enter a number
  - Converts it to an integer
  - Checks if it is **even** or **odd**
  - Prints a message indicating whether the number is even or odd
- Run the function **three times**, storing all numbers in a **list**
- After all three runs, print the list of numbers entered

**Example output**
```txt
Enter a number: 4
4 is even!
Enter a number: 7
7 is odd!
Enter a number: 10
10 is even!
All numbers entered: [4, 7, 10]
```

<Tabs items={['Hint 1', 'Hint 2']}>

<Tab>
Define a function using `def check_number():` and put all code inside it with proper **indentation**.  
You can append each number to a list after converting it to an integer.
</Tab>

<Tab>
Use `num % 2 == 0` to check if a number is even, and `else` for odd.  
Call the function **three times** in a row to let the user enter three numbers.
</Tab>

<Tab>
```python copy filename="number_checker.py"
# List to store numbers
numbers = []

def check_number():
    num = int(input("Enter a number: "))
    numbers.append(num)
    
    if num % 2 == 0:
        print(f"{num} is even!")
    else:
        print(f"{num} is odd!")

# Run the function three times
check_number()
check_number()
check_number()

# Show all numbers entered
print(f"All numbers entered: {numbers}")
```
</Tab>

</Tabs>


## Challenge 7

### Quiz üìù

**Goal:**  Practice **variables, input, conditionals, lists, and f-strings**.

**Requirements**
- Store the **questions** in a list
- Store the **correct answers** in a separate list
- Ask the user each question using `input()`
- Compare the answer using `if / else`  
- Print `"Correct!"` or `"Incorrect!"` for each answer
- Keep track of **how many answers are correct**
- Print the total correct answers at the end

**Example output**
```txt
Question 1: What is 5 + 3? 8
Correct!
Question 2: What is the capital of France? Paris
Correct!
Question 3: What color do you get when you mix red and white? Pink
Correct!
You got 3 out of 3 correct!
```

<Tabs items={['Hint 1', 'Hint 2']}>

<Tab>
Store all questions in a **list**, e.g.,  
`questions = ["What is 5 + 3?", "Capital of France?", "Red + White?"]`  
Do the same for answers in a separate list.
</Tab>

<Tab>
Access each question and answer by **index** (0, 1, 2).  
Compare using `if user_answer == answers[index]:`.
</Tab>

<Tab>
```python copy filename="quiz.py"
# Lists of questions and answers
questions = [
    "What is 5 + 3? ",
    "What is the capital of France? ",
    "What color do you get when you mix red and white? "
]

answers = ["8", "Paris", "Pink"]

# Keep track of score
score = 0

# Question 1
user_answer = input(questions[0])
if user_answer == answers[0]:
    print("Correct!")
    score += 1
else:
    print("Incorrect!")

# Question 2
user_answer = input(questions[1])
if user_answer == answers[1]:
    print("Correct!")
    score += 1
else:
    print("Incorrect!")

# Question 3
user_answer = input(questions[2])
if user_answer == answers[2]:
    print("Correct!")
    score += 1
else:
    print("Incorrect!")

print(f"You got {score} out of {len(questions)} correct!")
```
</Tab>
